******************* SWIFT User Documentation Instructions *******************

To access the documents included in this package, open the file index.htm with Internet Explorer.
Other browsers, such as, Firefox, Google Chrome ... are not supported but may work.

This package contains manuals in HTML and PDF format. To display the content,
you must have a recent web browser and PDF display software such as 
Adobe Reader (V7 or later). If you do not have this software installed, it can 
be downloaded free of charge from Adobe at http://www.adobe.com/ (follow the 
links to "Get Adobe Reader").

If you are displaying the PDF files within a web browser, make sure that 
you have the Adobe Reader plugin for your browser installed and activated.
