MT 601 Format Specifications

The format contains two sequences:

  • Sequence A General Information provides the details of the option confirmation.

  • Sequence B Reporting Information provides transaction information that must be reported to a trade repository. This sequence is optional and is only to be used for reporting to trade repositories, although some fields, for example, unique transaction identifier and prior unique transaction identifier, might be used on regular confirmations.
