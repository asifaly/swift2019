MT 537 Field Specifications

103. Field 98a: Date/Time

FORMAT

Option A :4!c//8!n (Qualifier)(Date)
Option C :4!c//8!n6!n (Qualifier)(Date)(Time)

PRESENCE

Optional in optional subsequence D1a1A1

QUALIFIER

(Error code(s): T89)

Order M/O Qualifier R/N CR Options Qualifier Description
1 O PRIC N   A or C Price Quotation Date/Time
2 O EXCH N   A or C Exchange Rate Quotation Date/Time

DEFINITION

This qualified generic field specifies:

EXCH Exchange Rate Quotation Date/Time Date/time at which the exchange rate was evaluated.
PRIC Price Quotation Date/Time Date/time at which the price was evaluated.

NETWORK VALIDATED RULES

Date must be a valid date expressed as YYYYMMDD (Error code(s): T50).

Time must contain a valid time expressed as HHMMSS (Error code(s): T38).
