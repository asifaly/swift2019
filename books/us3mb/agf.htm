  2. Cancellation of a Message

    The MT 392 can also be used for cancellation. The following issues however, have to be considered:

    • Acknowledgement: by sending the MT 392, the sender requests the receiver to cancel the confirmation previously sent. The receiver must acknowledge the request by sending an MT 396 back to the sender of the MT 392.

    • Routing: the MT 392 might not be routed to the same destination as the MT 360 to be cancelled.

    • Repetition of the original message: in the MT 392, only the sender's reference of the previous message is mandatory, all other fields are optional. When an MT 360 with CANC is used, the fields to be copied are mandated by the standard.
