MT 360 Field Specifications

15. Field 82a: Party A

FORMAT

Option A [/1!a][/34x]
4!a2!a2!c[3!c]		 (Party Identifier)
(Identifier Code)
Option D [/1!a][/34x]
4*35x		 (Party Identifier)
(Name and Address)

PRESENCE

Mandatory in mandatory sequence A

DEFINITION

This field identifies party A.

NETWORK VALIDATED RULES

Identifier Code must be a registered BIC (Error code(s): T27,T28,T29,T45).

USAGE RULES

For matching purposes, option A must be used when available.

Party A is either the sender :94A:BILA, or, the institution or corporate on whose behalf the message is sent :94A:AGNT, except when the Sender is a money broker :94A:BROK.

See the chapter Usage Rules for this MT.
