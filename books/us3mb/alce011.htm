MT 380 Field Specifications

11. Field 22H: Indicator: Buy/Sell Indicator

FORMAT

Option H :4!c//4!c (Qualifier)(Indicator)

PRESENCE

Mandatory in mandatory sequence B

QUALIFIER

(Error code(s): T89)

Order M/O Qualifier R/N CR Options Qualifier Description
1 M BUSE N   H Buy/Sell Indicator

DEFINITION

This qualified generic field specifies:

BUSE Buy/Sell Indicator Whether the message relates to the sale or purchase of financial instruments.

CODES

Indicator must contain one of the following codes (Error code(s): K22):

BUYI Buy The sender of the message is sending an order to buy.
SELL Sell The sender of the message is sending an order to sell.
