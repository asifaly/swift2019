MT 549 Field Specifications

6. Field 13A: Number Identification: Requested Message Type

FORMAT

Option A :4!c//3!c (Qualifier)(Number Id)

PRESENCE

Mandatory in mandatory sequence A

QUALIFIER

(Error code(s): T89)

Order M/O Qualifier R/N CR Options Qualifier Description
1 M REQU N   A Requested Message Type

DEFINITION

This qualified generic field specifies:

REQU Requested Message Type Number identifying the type of message or statement requested.

CODES

Number Id must contain one of the following codes (Error code(s): K13):

506 MT 506 Collateral and Exposure Statement
507 MT 507 Collateral Status and Processing Advice
509 MT 509 Trade Status Message
510 MT 510 Registration Status and Processing Advice
535 MT 535 Statement of Holdings
536 MT 536 Statement of Transactions
537 MT 537 Statement of Pending Transactions
538 MT 538 Statement of Intra-Position Advices
548 MT 548 Settlement Status and Processing Advice
558 MT 558 Triparty Collateral Status and Processing Advice
567 MT 567 Corporate Action Status and Processing Advice
569 MT 569 Triparty Collateral and Exposure Statement
575 MT 575 Report of Combined Activity
576 MT 576 Statement of Open Orders
586 MT 586 Statement of Settlement Allegements
