MT 506 Field Specifications

48. Field 12B: Type of Financial Instrument: Option Type

FORMAT

Option B :4!c/[8c]/4!c (Qualifier)(Data Source Scheme)(Type)

PRESENCE

Optional in optional subsequence C2

QUALIFIER

(Error code(s): T89)

Order M/O Qualifier R/N CR Options Qualifier Description
1 O OPTI N   B Option Type

DEFINITION

This qualified generic field specifies:

OPTI Option Type Specifies whether it is a call option or a put option.

CODES

If Data Source Scheme is not present, Type must contain one of the following codes (Error code(s): K12):

CALL Call Option Call option.
PUTO Put Option Put option.
